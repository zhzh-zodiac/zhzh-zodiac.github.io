> 初始化&一些基础命令

<!--more-->

### init hexo
```
// 安装主题 https://github.com/next-theme/hexo-theme-next
npm install hexo-theme-next

// 安装 https://github.com/theme-next/hexo-symbols-count-time
npm install hexo-symbols-count-time

// 安装 https://github.com/wzpan/hexo-generator-search
npm install hexo-generator-search --save

// 安装 https://github.com/next-theme/hexo-generator-searchdb
npm install hexo-generator-searchdb

// 安装deploy插件
npm install hexo-deployer-git
```
