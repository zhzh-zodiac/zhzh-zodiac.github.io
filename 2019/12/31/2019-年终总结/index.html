2019 年终总结

发表于 2019-12-31   |   分类于 个人随笔   |     |   阅读次数

2019年年初时定下的目标其实已经记不太清了，所以也预示着这一年是不达预期的一年啊。

下面还是根据这类结构简单的总结一下吧。

今年是意外事件多发的一年内，本来年初的计划是在老东家那里好好看，争取今年能有长足的进步。但是年中时意外被裁，被迫再一次的跳槽，所以造成很多计划都搁浅了。

干了什么

  • 换工作
  • 旅行桂林、大理
  • 相亲一次
  • 认识了一批很有正能量的创业人

关于换工作

上半年在老东家那本想努力工作在技术和职位上能有所提升，奈何公司业务发展不顺，在7月份时终于还是与老东家分手了。中途面试加休息一个月整后来到了新东家这边，然而可能因为面试不佳职级定的也不是很好。但因为这两年跳槽甚多，又是行业寒冬，遂不再挑选直接入职了。

在不可抗力的因素下，终究还是没能达到年初给自己定下的预期-成为小leader，提升自己的管理能力。但能在被裁后顺利过渡一方面有年轻的优势，一方面也是现在还无房贷、家庭的压力，但这些都是应该在考虑范围内，毕竟今年也有看房。平常需要做跟多准备来对抗这些不可抗力因素的影响。

关于旅行

在换工作期间还进行了一次长途旅行，短暂的游玩了桂林、阳朔、大理等地，见了毕业4年未见的同学朋友。被来和二水、攀哥计划的每年一次同行出游计划也因为他们二人的终身大事给耽误了，所以今年就趁换工作期间短暂的十来天来了场说走就走的旅行。一是让自己换个心情，二来也是能见见朋友见见祖国的山山水水。算是完成了自己每年出游的计划吧。

关于个人生活

在年初还回家相亲一次，见面后又在微信上聊天月余终是无法走到一起，就此别过。其实这次相亲也是临时由我姐通知我多，当然我知道他们肯定也是在长期打听的。因为春节不在老家过，所以在清明前告诉爷爷会在清明回家看看，然后在回家前一个周末被老姐告知回家相亲。因为从毕业开始就一直被亲戚朋友催婚，然后自己这几年在外也确实没找到自己的灵魂伴侣，就答应回家相亲试试。

然而我还是把家人们对相亲的态度想的太轻松了，自从相亲完后，两边的家长都是完全跟对方家长说自家子侄很满意，然后我家亲戚这边都是催促我主动点（因为异地），多找机会（五一之类的假期）约对方出来玩，但经过简单沟通后发现她还是对异地颇有顾忌，最终也没能成行。对方家长也是催的她不耐其烦，然后5.20那天她被朋友表白，第二天就跟我坦白来结束我们这段尴尬的关系，因为各种各样的关系，我们达成了一致意见。第一次相亲以失败告终。

关于交友

其实我觉得不能叫做交友，但也想不到啥更准确的词了就凑合着用了。在上半年关工作期间认识了叶叶、茶茶等年轻的创业者，从他们身上看到很多我不具备的特质，所以努力跟她们搭上关系，更多的了解她们创业经历。可能是因为自己心中也还藏着这样一个梦吧。

自我评价

总得来说今年算是诸事都不太顺利吧，遗憾颇多，但对自己而言也还是有所突破。虽然很多事没能完成，但也努力调和了自己的心态去迎接这个不那么完美的人生。

时间是单程的，所以不管今年好坏也都只能随风而去，以更加饱满的心态开始新的一年奋斗吧。

来年计划

  • 认真考虑可入手的楼盘，有可能的情况下开始作房奴吧
  • 在年底能晋升到P6
  • 训练工作外的技能：视频、PS、文章，每月完成两篇
  • 好好练习ukulele，做好5月年会的准备
  • 好好追下Yao_Kidly吧，不管结果如何
